🔗ally.map.keycode

Map of control event.keyCodes.

🔗Description

// example structure
var map = {
  // Element Focus
  tab: 9,

  // Navigation
  left: 37,
  up: 38,
  // ...
  home: 36,

  // Action
  enter: 13,
  escape: 27,
  space: 32,
}

🔗Usage

console.log(ally.map.keycode.enter);
