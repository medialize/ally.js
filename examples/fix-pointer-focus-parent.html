Fix Pointer Focus Parent Browser Bug

a11y.js provides the method a11y/fix-browser/pointer-focus to workaround an issue in Blink and WebKit causing focus given by MouseEvent and TouchEvent to be given to the next parent element accepting focus, rather than the element the event was dispatched to.

Usage

// load fixPointerFocusParent()
var fixPointerFocusParent = require('a11y/fix-browser/pointer-focus-parent');
// acquire context
var context = document.getElementById('element-to-fix');

// activate the fix on a specific sub-tree (for every browser)
var disengage = fixPointerFocusParent(context);

// activate the fix for the entire document (for every browser)
var disengage = fixPointerFocusParent();

// disable the fix
disengage();
